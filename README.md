# Смета материалов

Локальное Android-приложение для расчета отделочных материалов. Работает оффлайн, не требует входа, регистрации или админки, данные расчета не сохраняются после закрытия приложения.

## Что реализовано

- Экран ввода комнат с динамическим количеством комнат.
- Поля площади, длины, ширины, высоты потолков, внутренних и внешних углов.
- Добавление и удаление проемов с шириной и высотой.
- Режим шпаклевки: с потолком или без потолка.
- Автоматический расчет:
  - картон для пола = общая площадь пола * 1.2;
  - целлофан для окон = сумма площадей проемов * 1.2;
  - грунтовка = общая площадь пола / 6, округление до 1 знака;
  - шпаклевка = общая площадь пола * 10 или * 8.
- Предварительная смета с ручным добавлением материалов.
- Удаление только ручных позиций.
- Финальный просмотр и системная Android-кнопка поделиться сметой.

## Сборка

```bash
./gradlew assembleDebug
```

Debug APK:

```text
app/build/outputs/apk/debug/app-debug.apk
```
